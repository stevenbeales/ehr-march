Patient treatement API

Create patient diagnoses

POST /patient_treatments/create_diagnosis

Parameters

Name Description
auth_token Authentication Token
patient_id Patient id
diagnosis[diagnosis_code_id] Diagnosis code id
diagnosis[start_date] Start date
diagnosis[stop_date] Stop date
diagnosis[acute] Boolean, if true - acute
diagnosis[terminal] Boolean, if true - terminal
diagnosis[note] Note

Request

Headers

Content-Type: application/vnd.api+json

Route

POST /patient_treatments/create_diagnosis
