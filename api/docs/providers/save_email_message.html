Providers API

Save email message

POST /providers/save_message

Parameters

Name Description
auth_token Authentication Token
email_message[to_id] User id
email_message[subject_id] Subject id
email_message[body] Message
email_message[urgent] Boolean, if true - message is urgent
commit If "Send" message will be sent to real email

Headers

Content-Type: application/vnd.api+json

Route

POST /providers/save_message
