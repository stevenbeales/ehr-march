Sorter API

Sort appointments and paginate

GET /sorter/appointments

Parameters

Name Description
data[auth_token] Authentication Token
data[page] Pagination page number
data[field] Field order by, can be: "appointment_datetime_time", "patient", "appointment_type" other variants return all appointments
data[type] Type of order: "asc" or "desc"

Request

Headers

Content-Type: application/vnd.api+json

Route

GET /sorter/appointments?data[auth_token]=cc56f32a583c6f1a081dd266fee1e325&data[page]=0&data[field]=patient&data[type]=desc
