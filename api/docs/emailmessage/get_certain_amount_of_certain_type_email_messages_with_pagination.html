EmailMessage API

Get certain amount of certain type email messages with pagination

GET /email_messages/new_tab

Parameters

Name Description
data[auth_token] Authentication Token
data[amount] Amount of messages
data[page] Page pagination number
data[type] Type: inbox, sent, draft, archive or urgent

Headers

Content-Type: application/vnd.api+json
