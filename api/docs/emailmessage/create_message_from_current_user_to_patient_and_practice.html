EmailMessage API

Create message from current user to patient and practice

POST /email_messages/create_to_practice

Parameters

Name Description
auth_token Authentication Token
commit If commit = "Send", message will be sent to real email
email_message[patient_id] Patient id
email_message[practice_id] Practice id
email_message[subject_id] Subject id
email_message[body] Message
email_message[urgent] Boolean, if true - message is urgent

Request

Headers

Content-Type: application/vnd.api+json

Route

POST /email_messages/create_to_practice
