Ruby on Rails

- Ruby  2.3.0
- Rails 4.2.5.2
- RethinkDB 2.2.5
- Nobrainer
- Jsonapi-Resources
- Jsonapi-Serializers
- Devise-nobrainer
- Devise
- Devise-Authy
- Rspec_api_documentation

- Node.js 5.7.1

- Ember-cli 2.4.2
- Ember-Data 2.4.0
- Bootstrap 3.3.6
- Font-Awesome 4.5.0
- Liquid-Fire 0.23.0
- MongoDB or RethinkDB
